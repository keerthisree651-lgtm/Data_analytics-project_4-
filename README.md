# Data Analytics Project 4 - Power BI Data Visualization Dashboard

## Overview

This project demonstrates how raw e-commerce data can be transformed into meaningful business insights using Microsoft Power BI. The dashboard focuses on interactive visualizations, KPI tracking, and business performance analysis to support data-driven decision-making.

The project is designed to provide an executive-level overview along with operational insights through a clean, interactive, and professional dashboard.

---

## Objectives

- Build an interactive Power BI dashboard.
- Visualize key business metrics.
- Analyze sales performance and customer behavior.
- Monitor operational performance.
- Practice dashboard design and data storytelling.

---

## Tools Used

- Microsoft Power BI Desktop
- Microsoft Excel
- Data Visualization
- Dashboard Design
